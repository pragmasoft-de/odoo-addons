Werbeblocker

Die Werbung für SaaS von Odoo.com wird innerhalb von Odoo wird dauerhaft ausgeblendet. Außerdem wird ein "No Phoning home" aktiviert, welches ein regelmäßiges aber überflüssiges Signal deaktiviert.
